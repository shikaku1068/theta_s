Bookmarklets for RICOH THETA S

2016/02/11 - shikaku1068

In this page you can download bookmarklets to control RICOH THETA S remotely via a Web-browser.

There are three bookmarklets here: "LightweightShutter" makes it possible to repeat shooting at relatively short intervals, "3SecondsShutter" enables 3 seconds timer shooting and "VolumeChanger" provides control of the shutter sound's volume.

Precautions for use

  • Some browsers may fail to run the bookmarklets because of their length of code.
  • The user shall be responsible for any losses that may occur through the use of the bookmarklets downloaded from this site.

Usage

You need to install the bookmarklets in your browser to run them. If my explanation below isn't enough, please make reference to other sites and things about installation of a bookmarklet.

How to install

  1. Open a page of a bookmarklet you like from the next section.
  2. Add this page to the bookmark list in the usual way.
  3. Select the bookmarklet code displayed on this page from beginning to end and copy it to the clipboard.
  4. Open the editorial screen of the bookmark for this page registered above.
  5. Clear all the text in the URL box, paste the bookmarklet code from the clipboard into it and update the bookmark.

How to use

  1. Start your browser and open a dummy page ("about:blank" acceptable). (In some browsers opening a dummy page isn't necessary.)
  2. Connect your Android device with THETA S by Wi-Fi.
  3. Call the bookmarklet you installed in the previous subsection from the bookmark list. For the first time it will open a page on THETA S (http://192.168.1.1/osc/info/) and end.
  4. Call the bookmarklet again. The button to control THETA S will appear.

Chrome for Android doesn't allow a bookmarklet to be called from the bookmark list, but from the address bar. If using this browser, type the name of the bookmarklet into the bar, and the option to run it will be shown below.
