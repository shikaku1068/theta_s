Bookmarklets for RICOH THETA S

2016/01/25 - shikaku1068

In this page you can download bookmarklets to control RICOH THETA S via Web-browsers.

There are for now two bookmarklets here: “LightweightShutter” makes it possible to shoot at relatively short intervals and “3SecondsShutter” enables fixed 3 seconds timer shooting.

Precautions for use

  • Some browsers may fail to run the bookmarklets because of their volume.
  • The user shall be responsible for any losses that may occur through the use of the bookmarklets downloaded from this site.

Usage

You need to install bookmarklets in your browser to run them. Please make reference to other sites and things about installation of bookmarklets.
